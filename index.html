Projeto RPG - Stranger Things

Desenvolvido na disciplina de Estrutura de Dados Orientada a Objetos

Sobre o Projeto

Este projeto é um jogo de RPG baseado no universo de Stranger Things. O jogador controla um grupo de três personagens que enfrentam desafios sobrenaturais para impedir que Hawkins seja consumida pelo Mundo Invertido.

Objetivo do Jogo

O objetivo do jogo é derrotar monstros, resolver enigmas e coletar os Selos do Abismo para impedir que Vecna conquiste o mundo.

Funcionalidades

  • Acesso a um menu de instruções para entender o funcionamento.
  • Escolha de três personagens com atributos distintos.
  • Sistema de itens especiais que melhoram atributos.
  • Combate baseado em atributos como Força, Inteligência e Carisma.
  • Sistema de rolagem de dados (D20) para definir sucessos e falhas.
  • Progressão narrativa por quatro rodadas, com desafios diferentes.

Rodadas e Desafios

O jogo é dividido em 4 rodadas, cada uma contendo desafios específicos:

  • Rodada 1: Shopping Starcourt - Enfrente o Devorador de Mentes.
  • Rodada 2: Floresta das Trevas - Encontre os selos escondidos.
  • Rodada 3: Mundo Invertido - Supere as armadilhas de Vecna.
  • Rodada 4: Casa Creel - A batalha final contra Vecna.

Tecnologias Utilizadas

  • C++
  • Programação Orientada a Objetos
  • Estruturas de Dados
  • Operações CRUD
